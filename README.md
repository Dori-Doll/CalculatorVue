# Калькулятор на Nuxt UI

Це навчальний веб-калькулятор на Nuxt 4 і Nuxt UI.

## Можливості

- Додавання
- Віднімання
- Множення
- Ділення
- Виведення результату на екрані
- Спеціальний ефект: якщо результат дорівнює 0, фон сторінки починає безкінечно блимати (білий -> чорний -> білий) з інтервалом 0.5 секунди

## Технології

- Nuxt 4
- Vue 3
- Nuxt UI

## Встановлення

1. Відкрити термінал у папці проєкту.
2. Встановити залежності:

npm install

## Запуск у режимі розробки

npm run dev

Після запуску відкрийте адресу локального сервера, яку покаже Nuxt у терміналі.

## Продакшн збірка

npm run build

## Локальний перегляд продакшн-збірки

npm run preview

## Структура

- app/app.vue: інтерфейс калькулятора та логіка обчислень
- nuxt.config.ts: конфігурація Nuxt, підключення Nuxt UI

## Примітки

- Для обчислень використовується клієнтська логіка без бекенду.
- Анімація фону вмикається тільки коли результат строго дорівнює 0.
